# IT Ticket Analysis Project
## Tools Used
- Microsoft Excel
- Data Cleaning & Visualization
## Project Objective
To analyze IT ticket data and identify patterns, delays and efficiency issues.
## Key Insights
- Average daily ticket volume is 53.37 tickets
- Each agent handles approximately 1900-2000 tickets
- Average resolution time is 4.5 days
## Files Included
- Excel Dashboard 
- Project Presentation (PPT)
- Project Report (Word)
- Dashboard Screenshots
## Outcome
This project helps in understanding ticket trends and improving operational efficiency.
